# TaskPaper for VS Code

Simple task management extension for VS Code using Markdown.

## Features

- **Task Highlighting**: Automatically highlights task lines.
- **Completion Management**: Toggle task completion with a command.
- **Task Organization**: Side panel view to see all tasks, completed tasks, and today's tasks.
- **Tagging**: Add and remove tags easily.

## Usage

1. Open a `.md` file.
2. Create tasks using the format: `- [ ] Your task description`.
3. Use commands to manage your tasks:
    - `TaskPaper: Toggle Completion`
    - `TaskPaper: Add Tag`
    - `TaskPaper: Remove Tag`
    - `TaskPaper: Search Tag`

## Requirements

- VS Code 1.75.0 or higher.
