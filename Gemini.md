# Project overview
- このプロジェクトは、Visual Studio Code の拡張機能であり、シンプルなタスク管理ツールです。
- ユーザは markdown 形式でタスクを記載します。
    - 日付ごとに、前日のタスクファイルをコピーして新規にタスクファイルを作ることを想定しています。
    - ひとつのタスクファイルを長期に渡って編集し続けるような使い方は想定していません。
- 記載されたタスクをハイライト表示したり、タスク管理のための便利な機能を提供することが目的です。

# Detailed requirements
- 専用のモードを用意するのではなく、一般的な markdown の拡張に対して機能を追加する形で実現してください。
- ユーザが記載するタスクのテキストは examples/tasks.md に記載されています。
- 通常の markdown と同様に、タスクは入れ子構造にすることができます。
    - 入れ子になった行は折りたたむことができます。
- 各行は、タスク行かコメント行のいずれかで表されます。
    - タスク行の先頭には "[ ]" または "[x]" が付いており、タスクが完了すると "[x]" になります。
    - コメント行はタスク行とは異なり "[ ]" が付いていません。
- 各行には "@" で始まるタグをつけることができます。タグは複数つけることができます。
    - たとえば "@due(2026/01/31)" はタスクの期限を表します。
    - "@on(2026/01/31)" は、その日にタスクを実行しないといけないことを表します。
    - その他にもユーザは自由にタグをつけることができます。

## タスク行の詳細
- "[x]" が記載され完了されたタスクは、その行全体を打ち消し線で表示します。

## サイドパネルの詳細
- サイドパネルには、タスクの概要を表示します。
- サイドパネルは、全体のアウトライン、完了したタスク一覧、本日完了させないといけないタスク一覧、が表示されます。
    - 表示された内容をクリックすると、対応する行にジャンプします。

### オプション
- カレンダーのビューを追加して、どの日にタスクがあるかを表示する機能があってもよい。

## 追加コマンド
- タスクの完了・未完了をトグルする
- 特定のタグを付与する
- 特定のタグを外す
- 特定のタグを検索する
